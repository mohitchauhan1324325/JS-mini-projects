# 🚀 Mini Web Projects (HTML, CSS, JavaScript)

This repository contains a collection of **small projects built using HTML, CSS, and JavaScript**.
The goal of these projects is to practice core web development concepts and improve problem-solving skills by building real, working components.

---

## 📚 About This Repository

These projects focus on learning and experimenting with:

* HTML structure and semantic elements
* CSS styling and responsive design
* JavaScript DOM manipulation
* Event handling
* Animations and transitions
* API usage and asynchronous JavaScript

Each project is small but focuses on a specific concept to strengthen frontend development skills.

---

## 🧩 Projects Included

Some of the mini projects in this repository include:

* Flex Panels Image Gallery
* Random Dog Image Generator (API)
* Responsive Layout Practice
* Button Interaction Effects
* DOM Manipulation Examples
* JavaScript Array Exercises
* CSS Animation Experiments

More projects will be added as learning continues.

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript (ES6+)**

No external frameworks are used in most projects to keep the focus on **core web fundamentals**.
